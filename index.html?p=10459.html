    PHP and MySQL Web Framework Development

    Arun Rynet
    PHP
    Free
    • Description
    • Curriculum
    • Instructors
    This Course is SBL KHAS Claimable for Employers Registered with HRD Corp.

    DESCRIPTION

    This “PHP and MySQL Advanced” course, our PHP Certified Trainer will teach the intermediate to higher level PHP concepts, including a detailed introduction to object oriented programming, advanced language and web development methodologies like Web Serives, Security and PHP Frameworks development. Here is the list of outcomes of this course,

    • Utilize new PHP 7 concepts and Language Constructs.
    • Leverage object oriented programming (OOP) concepts in our applications.
    • Use Object Oriented Database Handling
    • Request a web service and process the return data.
    • Learn best practices & Security.
    • Build own Custom PHP framework similar to Laravel.
    • And much more..

     

    WHO IS THIS FOR

    • This this PHP and MySQL Advanced course is designed for PHP developers who have at least 6 months of PHP experience.

    WHAT WILL I LEARN

    • Introduction
    • PHP Syntax Review
    • Functions
    • Arrays
    • Web Features
    • Object Oriented Programming
    • Object Oriented Database Handling
    • PHP Data Objects(PDO)
    • Web Services
    • Project: Build your Own PHP Framework
    • Conclusion

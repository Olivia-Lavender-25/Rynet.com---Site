This Course is SBL KHAS Claimable for Employers Registered with HRD Corp.

DESCRIPTION

In this course, we shall begin by covering the basics of MongoDB and the key differences between NoSQL and SQL to help the participants build an intuitive understanding of foundational concepts for the course. This will be followed by engaging practical exercises to help the participants understand how to use database operations in the Mongo Shell. The participants use filtering, projection, sorting, and querying complex data structures to extract actionable insights from real-world datasets. At the end of the course, the candidate will build a fully functional CRUD WebApp using MongoDB & PHP.

WHO IS THIS FOR

This course is created for total beginners. Knowing SQL/MySQL basics is an added advantage to attend this course.

WHAT WILL I LEARN

  • Introduction
  • Installing MongoDB
  • Fundamentals of MongoDB
  • Mongo Shell
  • MongoDB Create & Read
  • MongoDB Update & Delete
  • GUI for MongoDB – COMPASS
  • Cloud MongoDB – Atlas
  • MongoDB – PHP driver
  • PROJECT – MONGOPHP CRUD APP
  • QA & Conclusion

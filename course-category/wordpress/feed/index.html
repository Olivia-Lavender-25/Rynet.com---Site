This Course is SBL KHAS Claimable for Employers Registered with HRD Corp.

DESCRIPTION

In this HRD Corp claimable Hands-On WordPress training course, we will show how to develop a website using WordPress with WordPress plugins like contact form, Booking Calendar, social sharing & gallery plugins. WordPress powers millions of websites and blogs. We will teach you to get the most out of the self hosted version of WordPress and create feature rich websites and blogs.

Our PHP Certified Trainer explain how to create posts and pages, customize the site with themes, widgets, and custom menus and extend even further with WordPress plugins. In addition to this, learn how to get more readers with social media sharing and comments, and how to adjust the settings that keep your site safe and secure.

WHO IS THIS FOR

Beginners with no knowledge of web-building. PHP/ MySQL would be an advantage.

WHAT WILL I LEARN

  • Introduction to WordPress CMS
  • Installing WordPress
  • WordPress Posts and Pages
  • WordPress User Profiles
  • Media and Images
  • Prepare to Build a Business Website
  • Introduction to WordPress Themes
  • Extend website Functionality Using Plugins
  • Move the Website from Production to Live Environment
  • Conclusion

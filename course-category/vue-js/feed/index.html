DESCRIPTION

In this practical Vue.js Building a Real-World Web Application training, our certified trainer will take you from novice to ninja in Vue.js, starting out with the very basics of Vue.js and then moving on towards creating fully functional Vue.js applications. Before we dive into Vue.js, the trainer teaches ES6+(Modern JS) features then dives into nuts and bolts of Vue and its ecosystem.

Apart from the basics, the trainer shall teach the candidates more techniques that are needed to ship vue.js applications. We cover the advanced topics like VueCli, Vue Router, Single File Components, Axios & API Calls. At the end of the course, the candidates will create a fully functional single-page application using Vue.js & Vue Eco System.

 

WHO IS THIS FOR

  • Knowledge with JavaScript will be an added advantage.
  • Basic HTML and CSS Knowledge is assumed throughout the Course

This Course is SBL KHAS Claimable for Employers Registered with HRD Corp.

DESCRIPTION

In this project based latest Angular training course, we shall teach you the core knowledge you need to deeply understand and build Angular applications along with the TypeScript. The candidates will learn about components, Directives, Modules, Services, Property and Event binding, routing and more. Along with a conceptual understanding of the framework, you will also gain direct experience with Angular by building a sample application.

As Angular  is backward compatible with the previous version, the candidates those who are interested to learn from Angular 2 to the latest version can also join this Angular course.

WHO IS THIS FOR

Angular 1 or Angular 2 knowledge is NOT required!
Basic HTML and CSS knowledge will be an added advantage
Prior JavaScript helps, but isn’t mandatory

WHAT WILL I LEARN

  • Intro to Angular 11
  • Dev Environment Setup (Node.js, Visual Studio Code & Git Bash)
  • Project setup using Angular CLI & Quickstart
  • Angular File Structure
  • TypeScript and its Features
  • Create Components using CLI & Manually
  • Modules and app.module.ts file
  • String Interpolation & Property Binding
  • Interfaces & Model Classes
  • ngIf and ngSwitch Conditional Directives
  • ngFor and Loops
  • Event Binding
  • Property Binding
  • ngClass & ngStyle
  • Introduction to Pipes
  • Forms & Events
  • Two Way Binding & ngModel
  • Services & Observables
  • Http Module
  • Angular Router
  • Course Project
